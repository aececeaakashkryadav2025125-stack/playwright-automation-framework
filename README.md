# 🚀 Playwright Automation Framework (UI + API)

## 📌 Overview
This project demonstrates end-to-end automation testing using Playwright with both UI and API testing.

---

## 🧰 Tech Stack
- Playwright (JavaScript)
- Node.js

---

## ✅ Features
- UI Automation (Login functionality)
- API Testing (GET & POST)
- HTML Test Reports
- Modular Folder Structure
- GitHub CI Integration

---

## 📂 Project Structure
tests/
  ui/      → UI Test Cases  
  api/     → API Test Cases  

---

## ▶️ Run Tests
npx playwright test

---

## 📊 View Report
npx playwright show-report

---

## 🎯 Key Highlights
- Automated UI testing using Playwright
- REST API validation using real endpoints
- Clean and scalable framework design
